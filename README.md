# fb-libarchive

The FreeBASIC Library Archive:
------------------------------
This Repository is a collection of precompiled libraries

GOALS: 
- offer lib versions, which match the translated header file versions, that are being shipped with the official FreeBASIC 1.10.1 release. 
- offer older/newer versions additionaly
- offer additionally well known libs which are NOT part of the official FreeBASIC 1.10.1 release
